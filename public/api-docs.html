Data sources

All data is sourced from public APIs and refreshed automatically. VigilMap normalises 8 sources into the unified VigilEvent schema.

🌋
USGS Earthquake Hazards
All earthquakes (24h) + significant events (7 days). Updates every 5 minutes.
🔥
NASA FIRMS
VIIRS satellite active wildfire detections. Near-real-time, 375m resolution.
🌩
NOAA NWS
US National Weather Service active alerts — tornado, hurricane, severe thunderstorm warnings.
⚔️
GDELT + News RSS
GDELT Doc API conflict events + BBC/DW/Guardian RSS feeds, merged and deduplicated.
🏥
WHO Disease Outbreak News
WHO official disease outbreak alerts, filtered for epidemiological relevance.
💰
ReliefWeb (UN OCHA)
UN humanitarian crisis reports covering food insecurity, economic shocks, and displacement. Updated daily.
✊
GDELT Economic
GDELT Doc API filtered for economic and labor keywords — sanctions, strikes, inflation, financial crises. Updates every 5 min.

Endpoints

GET /api/v1/events List current global events

Returns normalized VigilEvents from all active data sources. Cached for 5 minutes at the CDN edge.

Query parameters

Parameter Type Description
domain string Comma-separated domains: disaster, climate, health, conflict, economic, labor, science
severity string Minimum severity: info low medium high critical
limit integer Max results to return. Default + max: 100
since datetime ISO 8601 timestamp — only events after this time. Example: 2026-03-01T00:00:00Z

Example requests

# All events
curl https://vgilmap.vercel.app/api/v1/events

# High-severity conflict + health events
curl "https://vgilmap.vercel.app/api/v1/events?domain=conflict,health&severity=high&limit=20"

# Events in the last hour
curl "https://vgilmap.vercel.app/api/v1/events?since=$(date -u -v-1H +%Y-%m-%dT%H:%M:%SZ)"

Event schema

      
Every event from every source is normalized into the VigilEvent interface.


      

        

          id
          string
          Unique event ID, e.g. usgs-nc75209241
        

        

          timestamp
          ISO 8601
          Event occurrence time in UTC
        

        

          domain
          string
          One of: disaster, climate, health, conflict, economic, labor, science
        

        

          category
          string
          Specific type, e.g. earthquake, wildfire, outbreak
        

        

          severity
          string
          info · low · medium · high · critical
        

        

          title
          string
          Human-readable event title
        

        

          description
          string
          Short description (max 300 chars)
        

        

          location.lat
          number
          Latitude (WGS84)
        

        

          location.lng
          number
          Longitude (WGS84)
        

        

          location.label
          string
          Human-readable place name
        

        

          source
          string
          Data origin, e.g. USGS, WHO
        

        

          sourceUrl
          string
          URL to original event record
        

        

          confidence
          number
          0–1 confidence score (1.0 = authoritative sensor data)
        

        

          tags
          string[]
          Searchable keyword tags
        

        

          metadata
          object
          Source-specific extras, e.g. magnitude, frp, depth_km

Severity levels

      

        

          critical
          level 4
          Imminent threat to life or critical infrastructure
        

        

          high
          level 3
          Major event requiring immediate attention, M5+ earthquakes, large wildfires
        

        

          medium
          level 2
          Significant event, M3–5 earthquakes, active conflicts, severe weather
        

        

          low
          level 1
          Minor event, small earthquakes, routine weather alerts
        

        

          info
          level 0
          Informational — monitoring advisories, research alerts
        

      

    


    
    

      
Code examples


      
JavaScript / fetch

      

        
        
// Get all critical events
const res = await fetch(
  'https://vgilmap.vercel.app/api/v1/events?severity=critical'
);
const { data } = await res.json();
console.log(`${data.length} critical events`);

// Watch for new events every 5 minutes
setInterval(async () => {
  const since = new Date(Date.now() - 5 * 60000).toISOString();
  const r = await fetch(
    `https://vgilmap.vercel.app/api/v1/events?since=${since}`
  );
  const { data } = await r.json();
  // handle new events…
}, 5 * 60000);

      


      
Python

      

        
        
import requests

# Fetch high-severity conflict events
resp = requests.get(
    "https://vgilmap.vercel.app/api/v1/events",
    params={"domain": "conflict", "severity": "high", "limit": 20}
)
events = resp.json()["data"]

for ev in events:
    print(f"[{ev['severity'].upper()}] {ev['title']} — {ev['location']['label']}")

      


      
cURL

      

        
        
# Earthquake M4+ in the last 6 hours, formatted with jq
curl -s "https://vgilmap.vercel.app/api/v1/events?domain=disaster&severity=medium" \
  | jq '.data[] | {title, magnitude: .metadata.magnitude, location: .location.label}'

# Dashboard summary widget
curl -s https://vgilmap.vercel.app/api/v1/summary \
  | jq '{total: .total_events, by_domain}'
